## 添加数据源
在创建仪表盘之前，你需要先添加一个新的数据源，用来从数据库加载数据。

<TipCompat>注意：只有的<a class="underline" href="/docs-cn/advance-permissions#-1">Global Team</a>的管理员才能创建数据源</TipCompat>

1. 点击左侧菜单栏的 **配置管理 -> 数据源 -> 添加数据源**
2. 如果你已经搭建了prometheus数据库，就选择prometheus作为数据源，否则选择Static Data，该数据源是静态数据，一般用来测试用
3. 给你的数据源输入合适的名称，例如test(为了方便后续引用，文档中使用test作为名称)，如果开启名称旁边的默认按钮，则该数据源默认会作为所有图表的数据源，可以避免每次都显式的去为图表设定数据源，默认数据源同时只能存在一个
4. 如果是prometheus数据库，你需要输入promethues服务器的接口地址**HTTP -> URL**
5. 点击保存 & 测试，此时你应该看到成功的提示，如果失败且原因是**get xxx, dia tcp , connect refused**，那么应该是prometheus的地址配错了


## 查看已创建的数据源

1. 点击左侧菜单栏的 **配置管理 -> 数据源 **
2. 点击你想要查看的数据源卡片
