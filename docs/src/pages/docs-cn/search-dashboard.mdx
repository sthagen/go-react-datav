## 打开搜索面板
在任何页面都能使用快捷键o s(先按o再按s)来打开搜索面板，同时你可以点击左侧菜单栏的放大镜来打开

## 搜索仪表盘
在搜索面板的最上方输入关键字进行搜索，可以同时搜索仪表盘和文件夹

## 通过标签过滤
当你为仪表盘设置标签后，可以在搜索面的右上角，使用标签来过滤仪表盘
