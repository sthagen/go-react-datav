本章不再讲解仪表盘的创建、删除等基础功能，而是针对一些高级特性进行讲解。

## 图表相关
### 调整图表大小
将鼠标悬浮在图表上时，右下角会出现一个图标，拖动它即可调整。

### 隐藏图表标题
你知道该如何实现下面这种无标题的效果吗？
<img class="block mx-auto sm:mx-0 sm:flex-shrink-0 h-96" src="/img/docs/advance-dashboard-1.jpg" alt="Image" />

只要将图表的标题设置为空即可。

### 图表背景透明
在编辑页面 -> 设置中，开启`透明化`选项

## 仪表盘相关
### 隐藏顶部菜单
在仪表盘的设置 -> 通用设置中，关闭显示顶部菜单即可。

但是关闭后，问题就来了，顶部菜单不见了，还想开启怎么办？进不了设置页面了，不用着急，你只需要按下`d s`(先按d再按s)快捷键就可以打开仪表盘设置了。

### 自动保存
当你编辑了很多数据，但是页面突然崩溃，或者机器突然挂了，怎么办？在grafana你是没有任何办法的，但是datav有，只需要在
仪表盘的设置 -> 通用设置中开启自动保存选项，那么该仪表盘就会每隔10秒进行一次保存操作。

<TipCompat>注意，平时最好不要开启该选项，免得意外覆盖老的数据(很多时候，你只是想试试修改，而不是真的应用修改结果)，只有当你确认要进行长时间编辑时，再开启，使用完毕再关掉</TipCompat>

### 显示模式
仪表盘支持三种显示模式：普通模式、隐藏侧边菜单模式、TV模式，其中第三种模式在投影到tv时会非常有用，模式切换通过点击仪表盘右上角的tv按钮来完成，

### 自动刷新数据
很多时候，我们想要看到图表的数据随着时间进行自动查询和刷新，那么只要做到以下两步即可：
1.设置时间选择为`快速选择时间`，例如`Last 5 minutes`，总之不能选择一个固定时间段
2. 在时间选择右边的右边，将off换成一个时间间隔，例如`10s`

至此，图表就会每隔10秒自动刷新一次了。

### 分享URL
如果你想分享当前的图表数据给其它人该怎么做？点击仪表盘右上角的`+`，将时间、变量等信息添加到url中，再去分享。

### 修改仪表盘uid
在关联侧边菜单或者分享给别人时，一个好记的uid会更加合适，默认情况下图表的url会是这样：`http://localhost:3001/d/QLPzFK1Gz`，是的，最后那一串乱七八糟的就是当前的uid。

该如何修改呢：
1. 进入仪表盘设置 -> JSON元数据页面
2. 拖到最下面找到`"uid" : "QLPzFK1Gz"`
3. 修改为`"uid" : "test"`
4. 点击下方的保存修改

页面会自动刷新，并且更新url中的uid值。

## 仪表盘导入
手动创建仪表盘很有成就感，特别是看着漂亮图表被自己创建出来时，但是很多时候你不希望每次都从零创建一个仪表盘，这个时候就需要进行仪表盘导入了。

### 复制现有的仪表盘
进入仪表盘设置 -> JSON元数据页面，复制所有的json数据，这些就是我们仪表盘的元数据。

### 导入新的仪表盘
推出仪表盘设置页面，点击侧边菜单的**+ -> 导入**，将之前复制的仪表盘数据粘贴进去，点击提交，此时会看到一堆报错：
<img class="block mx-auto sm:mx-0 sm:flex-shrink-0 h-96" src="/img/docs/advance-dashboard-2.jpg" alt="Image" />

首先，修改一下名称，因为**仪表盘不能同名**, 其次修改uid，因为**uid不能重复**,当修改完成后，点击**Import**即可完成导入。
