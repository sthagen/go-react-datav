## 添加新用户
<TipCompat>你必须是Global团队的管理员，才能添加新用户，因为所有用户都是属于global团队的，一个用户可以同时归属于多个团队</TipCompat>

1. 在左侧菜单栏，选择**配置管理 -> 用户**，进入用户列表页面
2. 当前就你一个用户，过于孤单，现在来创建一个伙伴test，点击**添加用户**
3. 输入用户名test，密码和邮箱(邮箱可以暂时为空，后续再设置),选择global团队的角色，角色分为三种:
    - Viewer: 查看者，对于所属团队的仪表盘，只能进行查看，无法编辑保存(可能可以编辑，但是无法保存)
    - Editor: 编辑者, 对于所属于团队的仪表盘，可以查看和编辑保存
    - Admin: 管理员，可以任意查看、管理所属团队的仪表盘，同时可以管理团队自身
4. 为test用户选择Viewer角色
5. 点击提交，此时用户列表中多了一位小伙伴`test`

## 登陆test用户

登出当前用户admin，重新使用test进行登陆

1. 左边菜单栏选择**用户 -> 退出登录**
2. 输入test用户和对应的密码
3. 点击登陆

## 查看并编辑仪表盘
再让我们用test用户来查看和编辑下之前的仪表盘，还记得仪表盘的权限设置吧： 仪表板归属于test团队，global团队的成员只能查看，也就意味着
当前的test用户，只能查看仪表盘，但是无法编辑和保存，现在来验证下我们的猜测:
1. 打开搜索面板，进入仪表盘
2. 此时，已经能看到仪表盘，很好，查看权限有了
3. 点击仪表板的设置按钮，进去后，你会发现tab页全部消失了，只留下json数据
4. 重新回到仪表盘页面，点击保存

好了，无权限操作的提示，相信你已经看到，因此我们的猜测是对的：属于global团队的test用户，对于test团队的仪表盘，只有查看，没有编辑保存的权利。

那么接下来，就为test用户增加编辑保存的权限。

## 添加用户到团队
1. 退出test用户，重新使用admin进行登陆
2. 选择**配置管理 -> 团队**, 进入团队管理页面
3. 点击test查看进入团队详情页面
4. 点击添加成员(必须是test团队的管理员才能执行此操作)，弹出面板
5. 点击团队成员的选择框，选择test用户
6. 成员角色选择Editor
7. 点击提交

添加成功，在test团队详情页面已经能看到test用户，现在让再执行一次**查看并编辑仪表盘**，test用户已经可以成功的编辑仪表盘，并进行保存了。
