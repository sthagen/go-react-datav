团队是Datav中非常核心的概念，通过它，可以进行用户分组管理、权限管理、可见性设置，还可以自定义左边的菜单栏。

## 创建团队

<TipCompat>注意：只有的<a class="underline" href="/docs-cn/advance-permissions#-1">Global Team</a>的管理员才能创建团队</TipCompat>

1. 在左边菜单栏，选择**配置管理 -> 团队**，进入页面后，可以看到现有的团队列表,当前已经有了global团队，它是datav最核心的团队，所有的用户都会自动归属于该团队，
同时该团队的管理员就是datav的全局管理员，具有最高的权限
2. 点击新建团队，输入团队名称test
3. 点击提交

提交后，在团队列表中已经有了test团队，创建人是自己，点击test行的查看按钮，可以查看团队详情。

## 设置仪表盘归属
之前新建的仪表盘，默认是属于global团队，现在需要设置下，让它归属到新建的test团队。

1. 打开搜索面板，进入之前的仪表盘(如果你按照引导一步一步做到了现在，那该仪表盘是在test文件夹下)
2. 打开仪表盘的设置面板：使用快捷键d s(先按d，再按s)，或者顶部的设置按钮打开
3. 进入权限设置tab页，可以看到仪表盘当前是归属于global团队(正因为如此你才能编辑，因为你是global团队的管理员)
4. 将所有者设置为**test**，底下的**哪些团队可以查看**暂时无需修改
5. 点击更新按钮

刷新页面，你依然可以查看和编辑该仪表盘，因为你是test团队的创建者。
