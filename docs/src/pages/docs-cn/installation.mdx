## 下载并启动
### <Heading number="1">在Github下载服务</Heading>
访问<a href="https://github.com/opendatav/datav/releases" target="_blank">Github</a>下载最新版本的安装包



### <Heading number="2">启动服务</Heading>

```bash
## 进入到下载后的datav目录下
cd YOUR_DATAV_PATH

### 启动datav
./datav
```

服务器已经成功启动，访问http://localhost:3003，进行登陆，就能看到datav的UI页面。

<TipCompat>强烈建议使用Chrome或者Safari浏览器，IE/Edge可能会存在不兼容的问题</TipCompat>

```css

默认密码：admin
```


## 开发模式启动
如果你想对Datav进行二次开发，需要使用源码来启动。

### <Heading number="1">启动后端服务</Heading>
```bash
> git clone https://github.com/opendatav/datav
> cd datav
> go build
> ./datav generate ##只在初次启动或者安装了新插件后需要此步骤
> ./datav &
```

### <Heading number="2">启动UI服务</Heading>
```bash
```bash
> cd ui
> nvm use    
> yarn install
> yarn start
```
