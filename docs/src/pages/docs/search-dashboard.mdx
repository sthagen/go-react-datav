## Open the Search Page
Enter o s(enter o then s) to open the search page, you can also click on the search icon on the left menu panel.

## Search dashboard
Enter keywords on the top most search bar, you can search for folders and dashboards here.

## Filter by labels
If you have set labels for dashboards, you can filter dashboards by label on the top right corner.
